# 회의록

## 회의 개요
- 일시: 미확인 (녹취록에 명시되지 않음)
- 참석자: 발표자 1인 및 청중 (구체적 인원 미확인)
- 주제: AI 및 자동화 기술을 활용한 디지털 사업본부 업무 자동화 방안 검토 및 데모 시연

---

## 주요 논의 사항

### 안건 1: E-book 팀 GA4 데이터 분석 자동화

- **현황 및 문제점**
  - E-book 팀은 Google Analytics 4(GA4) 데이터를 매번 웹페이지에 직접 접속하여 수동으로 그래프 및 데이터를 확인·정리하고 있으며, 이에 따른 업무 공수가 상당함

- **자동화 방안**
  - GA4는 Google에서 API를 공식 제공하므로, 해당 API를 통해 데이터를 자동 수집 가능
  - 수집된 데이터를 LLM에 전달하여 트렌드 분석, 이탈 구간 분석, 리포트 생성 등을 자동화할 수 있음
  - 분석 결과가 만족스럽지 않을 경우, LLM에 가이드라인을 제공하여 결과를 지속적으로 개선 가능
  - 사내 미니 PC 또는 소형 서버를 활용한 폴링(Polling) 방식으로 5~10분 단위 실시간 모니터링 구현 가능
  - 이상 감지 시 Slack 또는 Telegram을 통한 실시간 알림 발송 가능

- **데모 시연 내용 (SNDWorks 운영 커뮤니티 사이트 'Symbolica' GA4 데이터 활용)**
  - MCP(Model Context Protocol)를 통해 GA4 속성에 연결
  - 세션, 사용자 수, 참여율, 체류 시간 등 주요 지표 자동 분석 및 그래프 생성
  - 유입 채널별 성과 분석, 콘텐츠·검색 트렌드 분석, 신규/기존 고객 행동 비교, 이벤트 페이지별 전환율, 이탈 구간 분석 등 종합 리포트 자동 생성 확인

### 안건 2: 크레마 클럽 팀 업무 자동화

- **현황 및 문제점**
  - 경쟁사 베스트셀러 수동 조회, 앱 별점·리뷰 수동 수집, 일/월 매출 리포트 수동 작성 등 반복적인 수작업으로 인한 업무 공수 발생

- **자동화 개념 구분**
  - **LLM 개입 불필요한 자동화**: 도서 리스트 추출, 별점 수집 등 단순 반복 작업은 사전 프로그래밍된 스크립트로 자동 실행 가능
  - **LLM 개입이 필요한 자동화**: 수집된 데이터를 바탕으로 리포트 작성, 구매자 행동 분석 등 고도화된 분석 작업에 LLM 활용

- **데모 시연 내용**

  1. **경쟁사 베스트셀러 웹 크롤링 자동화**
     - 알라딘, 교보문고 등 경쟁사의 주간 베스트셀러 목록을 자동 크롤링하여 엑셀 파일로 저장
     - 커맨드 한 번 실행으로 전체 리스트 자동 수집 및 날짜별 업데이트 가능 확인

  2. **앱 리뷰 자동 수집**
     - iOS 앱스토어에 직접 접속하지 않고도 앱 이름, 최근 리뷰 건수, 평점 등을 자동으로 수집 가능 확인

  3. **크레마 머니 월간 자동 리포트 생성**
     - MSTR 등 데이터 소스에서 추출한 엑셀 파일을 입력값으로 사용
     - 사전 정의된 스킬(업무 가이드라인 포함)을 통해 AI가 자동으로 CP 정산 현황 분석, 도서별 상세 데이터 정리, 대시보드 및 그래프 포함 월간 리포트 자동 생성
     - 기존 수작업(VLOOKUP 등 엑셀 함수 활용) 대비 업무 효율 대폭 향상 가능 확인

- GA4 데이터 분석 자동화 및 크레마 클럽 팀 업무 자동화 모두 현재 기술 수준에서 구현 가능함을 데모를 통해 확인
